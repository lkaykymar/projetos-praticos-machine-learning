# projetos-praticos-machine-learning
Repositório destinado aos meus estudos práticos de Machine Learning. Aqui reúno notebooks aplicando diferentes algoritmos de classificação, regressão e agrupamento, explorando desde o pré-processamento dos dados até a avaliação dos modelos utilizando conjuntos de dados reais.

## estudos

📘 naive_bayes_credit_risk.ipynb

Aplicação do algoritmo Gaussian Naive Bayes em um problema de classificação de risco de crédito. O estudo contempla análise exploratória dos dados, tratamento de valores inconsistentes, padronização das variáveis, treinamento do modelo e avaliação por meio de métricas como matriz de confusão e acurácia.
